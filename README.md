# Joan Ajifa Usman Analytics Portfolio

#[Project : Human Resource Analytics Dashboard](https://github.com/Joanajifa/Human-Resource-Analytics-Dashboard)
Developed an interactive HR Analytics Dashboard in Microsoft Excel to analyze workforce demographics, employee performance, engagement levels, satisfaction metrics, and training effectiveness. 
The project features KPI cards, Pivot Tables, Pivot Charts, Slicers, and calculated columns to provide a comprehensive view of workforce trends and organizational performance.

**Tools Used:** Excel, Pivot Tables, Pivot Charts, KPI Cards, Slicers, Conditional Formatting

**Key Features:**
- Workforce Overview Dashboard
- Performance & Engagement Dashboard
- Training & HR Investment Dashboard
- Interactive filtering with Slicers
- Automated KPI reporting
